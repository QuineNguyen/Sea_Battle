# Sea Battle

Sea Battle is a two-player Battleship game built with Java. The project follows a client-server architecture, where players connect to a central server, join a match, place ships on a 10x10 board, and take turns attacking the opponent's grid until one side wins.

## Features

- Client-server multiplayer gameplay
- Player matchmaking and waiting room UI
- Ship placement on a 10x10 board
- Turn-based attacks with hit/miss feedback
- Win, lose, and timeout notifications
- Basic chat and invite-related game flows

## How It Works

1. Start the server.
2. Launch one or more clients.
3. Connect to the server and enter a player name.
4. Invite or join another player.
5. Place ships and begin the match.

## Configuration

The default server port is `6969`, and the board size is `10 x 10`.
